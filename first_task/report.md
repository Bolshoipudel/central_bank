## 3. Расхождения между источниками и их обработка

### Правило выбора итоговой ставки

1. Сравниваются только **фиксированные** ставки 
2. Все источники совпадают → ставка итоговая
3. Расхождение ≤ 0.5 п.п. → если есть информация с официального сайта, берем ее, иначе среднее арифметическое по агрегаторам
4. Расхождение > 0.5 п.п. → «противоречие», приоритет: офиц. сайт > banki.ru > sravni.ru
5. Данные из одного источника → принимаются как есть
6. Фикс. ставка > 15.0% на 3 года → проверка (возможно, ставка плавающая)

## 4. Использование LLM в решении

### Роль LLM на каждом этапе

#### 4.1. Парсинг / извлечение данных

- **Проблема:** Агрегаторы (sravni.ru, banki.ru) рендерят данные через JavaScript — 
  классический HTTP-парсинг (requests + BeautifulSoup) не работает
- **Решение:** LLM (Claude) анализировал скриншоты веб-страниц и извлекал 
  структурированные данные: название продукта, ставку, срок, тип продукта
- **Пример:** Из скриншота страницы sravni.ru для Сбербанка LLM извлёк 7 продуктов 
  с названиями, ставками на 6 мес / 1 год / 3 года и метками (накопительный счёт vs вклад)

#### 4.2. Нормализация

- **Классификация продуктов:** LLM различала «вклад» от «накопительного счёта» по 
  меткам на скриншотах и характеристикам (пополнение/снятие, срок)
- **Тип ставки:** LLM определяла фиксированную vs плавающую ставку по описанию условий 
  (например, «ставка = ключевая ставка ЦБ − 2 п.п.» → плавающая)
- **Унификация названий:** «МКБ», «Московский Кредитный Банк», «Московский Кре...» → единое «МКБ»

#### 4.3. Валидация и проверка

- **Обнаружение аномалий:** LLM выявила, что продукты Сбербанк «Ключевой» (15.8%) и 
  Альфа-Банк «Ключевой» (16.53%) имеют плавающие ставки, хотя визуально 
  отображались как обычные вклады
- **Причины расхождений:** LLM объяснила разницу в ставках Газпромбанка 
  (5.3% vs 8.3%) наличием бонуса за опцию «Накопления» на banki.ru
- **Проверка разумности:** Сравнение ставок с ключевой ставкой ЦБ (15.0%) — 
  фиксированные ставки 6-12% на 3 года адекватны при ожидании снижения ставки

## 5. Кейсы: обоснование выбора ставки для 3 банков

### Кейс 1: СберБанк

**Итоговая ставка:** 6.5% (Лучший % (в конце срока))

**Как система пришла к значению:**

1. **sravni.ru** показал 5 вкладов. Лучшая фикс. ставка на 3 года — 6.5% 
   (продукты «Лучший %» и «СберВклад»)
2. **banki.ru** показал вклад «Ключевой» со ставкой 15.8% на 3 года
   Однако оказалось, что ставка плавающая и равна текущей ключевой ставке минус 2 п.п., 
   т.е. при снижении ключевой ставки доходность упадёт
3. **Решение:** Итоговая фикисрованная ставка = 6.5%

### Кейс 2: Россельхозбанк

**Итоговая ставка:** 8.2% (Свой вклад)

**Как система пришла к значению:**

1. **sravni.ru** показал «Свой вклад» со ставкой 8.2% на 3 года
2. **banki.ru** показал тот же продукт «Свой вклад» со ставкой 7.8%
3. **Офиц. сайт (rshb.ru)** показал «Свой вклад» со ставкой 8.2% 
   (калькулятор: 1 млн ₽, 36 мес, без пополнения/снятия, «Новый сберегатель»)
4. Расхождение = 0.4 п.п. (≤ 0.5 п.п.) → «незначительное расхождение»
5. **Причина расхождения:** banki.ru, вероятно, отображает устаревшую ставку 
   или ставку без опции «Новый сберегатель»
6. **Решение:** Итоговая ставка = ставка с официального сайта = 8.2% 
   (офиц. сайт приоритетнее агрегаторов как первоисточник)

### Кейс 3: Газпромбанк

**Итоговая ставка:** 8.3% (Новые деньги)

**Как система пришла к значению:**

1. **sravni.ru** показал несколько вкладов. Лучшие — «Новые деньги» и «Копить» 
   по 8.3% на 3 года. «В Плюсе» — 5.3%
2. **banki.ru** показал «В плюсе» со ставкой 8.3%, но с примечанием: 
   базовая ставка 5.3% + бонус 3% за опцию «Накопления» (требует мин. остаток 
   500 000 ₽/мес на накопительных счетах или подписку Газпром Бонус)
3. **Официальный сайт**: не доступен калькулятор по вкладам
3. **Расхождение** sravni/banki по «В Плюсе»: 5.3% vs 8.3% (3 п.п.) — 
   **противоречие**, но вызвано разным учётом бонусов, а не ошибкой данных
4. **Решение:** Итоговая ставка = 8.3% (по продуктам «Новые деньги» / «Копить» 
   на sravni.ru, которые дают ту же ставку без дополнительных условий)

## 6. Сравнение с другими LLM: типичные ошибки ИИ при сборе ставок

Для оценки качества подхода было проведено сравнение с результатами двух других LLM-агентов
(GigaChat от Сбера и Алиса от Яндекса), которым был дан аналогичный промпт

### 6.1. GigaChat (Сбер)

**Основные ошибки:**

| Банк | GigaChat | Реальность | Проблема |
|------|----------|------------|----------|
| Альфа-Банк | «Максимальный доход» 13,5% | Альфа-Вклад **7,30%** | Выдуманное название, ставка завышена в 1.8 раза |
| Т-Банк | «Надежный» 12,5% на 3 года | **Нет вкладов на 3 года** | Выдуманный продукт, несуществующий срок |
| Газпромбанк | «Ключевой момент» 17% фикс. | **Такого продукта нет** | Галлюцинация |
| СберБанк | «СберВклад» 12,06% | СберВклад **6,5%** | Ставка завышена в 1.85 раза |
| Россельхозбанк | «Премиум» 13% | Свой вклад **8,0%** | Выдуманное название, ставка завышена |
| МКБ | «Оптимальный МКБ» 13,5% | МКБ. Гранд **12,1%** | Выдуманное название |
| Совкомбанк | «Золотой стандарт» 13,5% | Весенний доход **12,0%** | Выдуманное название и неверная ставка|

**Итог:** GigaChat выдумывает названия продуктов и генерирует некорректные
ставки. Ни один из продуктов выше не существует

### 6.2. Алиса (Яндекс)

Алиса показала более качественные результаты: названия продуктов в основном реальные,
Т-Банк и Райффайзенбанк определены корректно. Однако **ключевая ошибка** — завышенные ставки:

| Банк | Алиса | Реальность | Причина расхождения |
|------|-------|------------|---------------------|
| Альфа-Банк | «до 14%» | **7,30%** (офиц. сайт) | Ставка 14% — для срока 3-4 мес, не 3 года |
| СберБанк | «до 14%» | **6,5%** | 14% — для 6 мес на «новые деньги» |
| ВТБ | «до 14,3%» | **11,5%** | 14,3% — для 61 дня |
| Газпромбанк | «до 14%» | **8,3%** | 14% — для 120 дней |
| Россельхозбанк | «до 13,4%» | **8,0%** | «Ультра Доходный» требует от 1,5 млн руб. |
| МКБ | «13,7%» | **12,05%** | «Простая выгода» нет на 3 года |
| Совкомбанк | «до 14,5%» | **12,0%** | С «Халвой», вероятно короткий срок |
| Т-Банк | макс. 24 мес | макс. **24 мес** | Срок указан корректно (единственный случай) |

**Итог:** Алиса не фильтрует ставки по конкретному сроку. Формулировка «до X%» берётся
из рекламных заголовков банков, где всегда указан максимум по короткому сроку (несколько месяцев)

## 7. Заключение

1. **Главная ошибка агентов** — использование ставок «до X%» без привязки к сроку 3 года.
   Банки рекламируют максимальные ставки (для сроков 3-6 мес), а LLM копируют их
2. **Единственный надёжный путь** — визуальная проверка через калькулятор на сайте банка
   или агрегатор с фильтром по сроку + анализ скриншотов с помощью LLM
3. **LLM полезен не для генерации данных, а для их обработки**: извлечение из скриншотов,
   классификация (вклад vs счёт, фикс. vs плавающая), нормализация и перекрёстная проверка
4. **Два банка не предлагают 3-летних вкладов**: Т-Банк (макс. 24 мес) и 
   Райффайзенбанк (вклады нельзя открыть с 2024 года)
5. **Расхождения между агрегаторами** вызваны разным учётом бонусных опций, 
   различием продуктовых каталогов, условиями «для новых клиентов»
