# Проект: Сбор ставок по вкладам ФЛ — топ-10 банков России

## Описание задачи

Задание 1 тестового задания ЦБ. Нужно собрать и структурировать ставки по вкладам на 1 млн руб.
сроком на 3 года для 10 крупнейших банков РФ, используя минимум 2 источника.

## Текущее состояние — ОСНОВНАЯ РАБОТА ВЫПОЛНЕНА

### Что сделано:

1. **Данные собраны** из 3 источников (скриншоты от пользователя → извлечение LLM):
   - `data/source_sravni.json` — sravni.ru (24 записи, все 10 банков)
   - `data/source_banki.json` — banki.ru (10 записей)
   - `data/source_official.json` — офиц. сайты (2 записи: Альфа-Банк, ДОМ.РФ)

2. **Python-скрипт** `deposit_rates.py` — загружает JSON, нормализует, перекрёстно проверяет,
   генерирует `report.md`. Запуск: `/c/Python_Environ/python.exe deposit_rates.py`

3. **Отчёт** `report.md` — полный отчёт с таблицами, расхождениями, кейсами, анализом LLM-ошибок.

  МКБ, Совкомбанк) — пользователь может предоставить скриншоты калькуляторов
- Расширить анализ плавающих вкладов (Сбер «Ключевой» 15.8%, Альфа «Ключевой» 16.53%)
- Сделать визуализацию (графики сравнения ставок)

## Технические детали

- Python: `/c/Python_Environ/python.exe`
- Кодировка: скрипт содержит фикс для Windows cp1251 (sys.stdout.reconfigure)
- Сайты банков рендерят через JS — парсинг невозможен без headless-браузера
- LLM-агенты (GigaChat, Алиса) не умеют фильтровать по сроку → дают завышенные ставки
