# Задание 3: Топ-10 банков по прибыли за май 2021 (форма 101 ЦБ РФ)

## Что делает решение

1. Скачивает с сайта ЦБ РФ архивы формы 101 (помесячные данные о балансах банков)
   и формы 102 (поквартальный отчёт о финансовых результатах).
2. Рассчитывает по каждому банку:
   - абсолютную чистую прибыль за май 2021,
   - прирост прибыли за месяц (май 2021 vs апрель 2021),
   - прирост прибыли за год (май 2021 vs май 2020).
3. Выгружает все банки на лист `data` итогового Excel + создаёт четыре
   **именованных диапазона** — это контракт между Python и Excel, поверх которого
   дальше собирается дашборд:
   - `bank_names`  — названия банков (столбец B),
   - `profit_abs`  — абсолютная прибыль за май 2021 (C),
   - `profit_mom`  — MoM прирост (F),
   - `profit_yoy`  — YoY прирост (G).
4. Сверяет расчёт с формой 102 за Q1 2021 и выводит сводку на лист `validation`.
5. **Интерактивный дашборд** (выпадающий список с выбором показателя, Top-10
   и горизонтальная диаграмма) собирается в Excel вручную поверх именованных
   диапазонов. Пошаговая инструкция — ниже в разделе «Сборка дашборда».

Такое разделение — осознанное: тяжёлая часть (DBF, деаккумуляция YTD, сверка
с формой 102) автоматизирована в Python, а сама визуализация делается средствами
Excel — так проще подогнать оформление под вкус и продемонстрировать работу
с формулами `LARGE` / `INDEX` / `MATCH` / `CHOOSE`, Data Validation и BarChart.

## Как запустить

```bash
pip install -r requirements.txt

# 1) Один раз — качаем данные с ЦБ (нужен интернет + утилита bsdtar для .rar).
python download_data.py

# 2) Обработка и сборка Excel — работает офлайн.
python process_form_101.py
```

Результат: `bank_profit_report.xlsx` рядом со скриптами.

## Зависимости среды

- Python 3.10+.
- Утилита `bsdtar` (часть libarchive). На macOS доступна из коробки в `/opt/anaconda3/bin/`
  или ставится через `brew install libarchive`. На Ubuntu: `sudo apt install libarchive-tools`.
- Из Python-пакетов — см. `requirements.txt`:
  - `dbfread` — чтение DBF (чистый Python, cp866/cp1251),
  - `openpyxl` — запись .xlsx вместе с формулами, data-validation и диаграммами,
  - `requests` — только для `download_data.py`.

## Методика

### Расчёт чистой прибыли за месяц (форма 101)

Форма 101 содержит **накопительные (YTD) значения с начала года** по всем балансовым
счетам. Чистая прибыль банка за период вычисляется как:

    прибыль_YTD = Σ IITG по счетам 70601–70605, 70613, 70615
                −  Σ IITG по счетам 70606–70611, 70614, 70616

Отсутствие у банка отдельных счетов из списка — норма: суммируем только то, что есть.

Месячная прибыль получается деаккумуляцией соседних YTD:

    прибыль_май_2021 = YTD(01.06.2021) − YTD(01.05.2021)
    прибыль_апр_2021 = YTD(01.05.2021) − YTD(01.04.2021)
    прибыль_май_2020 = YTD(01.06.2020) − YTD(01.05.2020)

Отсюда:

    MoM прирост = прибыль_май_2021 − прибыль_апр_2021
    YoY прирост = прибыль_май_2021 − прибыль_май_2020

### Валидация через форму 102

Форма 102 — поквартальный отчёт о фин. результатах, данные тоже YTD. Для Q1 2021
прибыль считается как Σ(код 61101) − Σ(код 61102) по колонке `SIM_ITOGO`.
Этот результат для каждого банка должен совпадать с YTD-прибылью по форме 101 на
01.04.2021.

На текущих данных:
- 383 из 392 банков совпали до копейки,
- 4 банка отсутствуют в форме 102 (их не сверяем),
- 5 банков расходятся более чем на 1 %. Max абс. расхождение — 240,5 млн ₽ (REGN 1557);
  вероятно, из-за реорганизаций и отзыва лицензий в течение квартала.

Итог: метод подсчёта по форме 101 даёт корректный результат для подавляющего
большинства банков.

### Единицы измерения

В форме 101 `IITG` хранится в **тысячах рублей**. В итоговом Excel все значения
приведены к **миллиардам рублей** (деление на 1 000 000) — это привычный масштаб
для банковских прибылей (Сбер за май 2021 — около 102 млрд ₽).

## Сборка дашборда (в Excel, руками, ~5 минут)

Всё, что нужно, уже лежит на листе `data` — включая четыре именованных диапазона.
Дашборд собирается поверх них.

### 1. Новый лист `dashboard`

В книге `bank_profit_report.xlsx` создать пустой лист `dashboard` и перетащить
его первым.

### 2. Выпадающий список (ячейка `C3`)

Вписать туда начальное значение — например, «Абсолютная прибыль за май 2021».
Затем: **Data → Data Validation → List → Source**, и в поле источник указать
три варианта через `;` (в русской локали Excel разделитель именно `;`):

```
Абсолютная прибыль за май 2021;Прирост прибыли за месяц (май 2021 vs апрель 2021);Прирост прибыли за год (май 2021 vs май 2020)
```

### 3. Индекс варианта (ячейка `C4`, можно спрятать серым)

```
=MATCH(C3, {"Абсолютная прибыль за май 2021";"Прирост прибыли за месяц (май 2021 vs апрель 2021)";"Прирост прибыли за год (май 2021 vs май 2020)"}, 0)
```

Результат — 1, 2 или 3. Это индекс для `CHOOSE`.

### 4. Top-10 таблица (строки 7–16, колонки B/C/D)

Шапка в строке 6: `Место | Банк | Значение, млрд ₽`.

Для строки `r` (от 7 до 16):

| Ячейка | Формула                                                                                                             |
|--------|---------------------------------------------------------------------------------------------------------------------|
| `Br`   | `=r-6`   (просто числа 1…10)                                                                                        |
| `Dr`   | `=LARGE(CHOOSE($C$4, profit_abs, profit_mom, profit_yoy), B{r})`                                                    |
| `Cr`   | `=INDEX(bank_names, MATCH(D{r}, CHOOSE($C$4, profit_abs, profit_mom, profit_yoy), 0))`                              |

Формат колонки D — `#,##0.00;-#,##0.00;-`. При смене `C3` таблица пересчитается.

### 5. Горизонтальная диаграмма

Выделить диапазон `C6:D16` (с заголовками) → **Insert → Bar Chart → Clustered Bar**.

Полезные штрихи:

- **Format Axis (ось категорий) → Categories in reverse order** — чтобы #1 был сверху.
- **Убрать легенду** — серия одна.
- **Format Data Labels → Value, Position = Outside End, Number format = `0.0`** — подписи у концов полос, одним знаком после запятой.
- Заголовок графика — «Топ-10 банков».

Диаграмма построена на том же диапазоне `C6:D16`, поэтому при смене `C3`
она автоматически обновляется вместе с таблицей.

### 6. Проверка

Поменять значение в `C3` между тремя вариантами — таблица и диаграмма должны
реагировать. В первом режиме лидер — Сбербанк (~102 млрд ₽), во втором и третьем
порядок заметно другой.
